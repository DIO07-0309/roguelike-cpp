━━━ Quick Build ━━━

Prerequisites:
  • CMake 3.16+
  • Visual Studio 2022 (MSVC) —OR— MinGW-w64 + g++

Steps (MSVC):
  1. Open "Developer Command Prompt for VS 2022"
  2. cd C:\Users\HP\Desktop\Roguelike-CPP版
  3. cmake -B build -G "Visual Studio 17 2022"
  4. cmake --build build --config Release
  5. copy build\Release\roguelike_cpp.exe .

Steps (MinGW):
  1. cmake -B build -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release
  2. cmake --build build
  3. copy build\roguelike_cpp.exe .

━━━ Mod Support ━━━

  Place mod folders in mods/<name>/ with mod.json:
    {
      "id": "my_mod", "name": "My Mod", "version": "1.0",
      "priority": 100, "provides": ["boss", "enemy"]
    }

  See docs/G4_PLATFORM_BIBLE.md for full Mod API.
